# Active Directory Attack and Defense Lab

## Contexto
Proyecto académico grupal enfocado en la simulación de ataques a un entorno Active Directory
y su posterior detección y mitigación.

## Rol personal
- Ejecución de ataques desde Kali Linux
- Análisis de eventos en Wazuh
- Propuesta de medidas de mitigación

## Técnicas utilizadas
- Credential access
- Lateral movement
- Atomic Red Team simulations

## Detección
Se identificaron eventos anómalos mediante reglas personalizadas en Wazuh.

## Mitigación
- Hardening de políticas de AD
- Ajustes de monitoreo
- Revisión de permisos

## Aprendizaje
Se reforzó el entendimiento de AD, detección basada en logs y respuesta a incidentes.
