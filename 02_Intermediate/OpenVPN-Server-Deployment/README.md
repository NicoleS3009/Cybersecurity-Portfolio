# Montaje de Servidor OpenVPN

## Contexto
Laboratorio académico enfocado en la implementación de una VPN segura
para acceso remoto a un entorno controlado.

## Objetivo
Implementar un túnel cifrado para proteger las comunicaciones entre clientes
y el servidor.

## Tecnologías
- OpenVPN
- TLS
- Certificados digitales
- Linux

## Implementación
- Configuración del servidor OpenVPN
- Generación y gestión de certificados
- Configuración de clientes

## Pruebas
Se validó la conexión segura y el cifrado del tráfico mediante pruebas de conectividad.

## Seguridad
- Uso de certificados para autenticación
- Cifrado de tráfico
- Restricción de accesos

## Aprendizaje
Comprensión práctica de VPNs, criptografía aplicada y seguridad de red.
